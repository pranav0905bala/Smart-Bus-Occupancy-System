# Smart-Bus-Occupancy-System


## Overview
This project developed a microcontroller-based smart bus occupancy tracking system using IR sensors to monitor passenger movement and provide real-time occupancy information.

## Problem Statement
Public transportation systems often lack accurate real-time occupancy monitoring. The goal was to develop a reliable and low-cost system for tracking passenger entry and exit.

## System Architecture
- IR sensors for passenger detection
- Arduino microcontroller for processing
- Occupancy counting logic for real-time updates

## Tech Stack
- Arduino
- IR Sensors

## Implementation
- Designed passenger detection logic using IR sensors
- Developed embedded control logic for counting passengers
- Implemented real-time occupancy tracking using Arduino

## Results
- Reliable passenger counting system
- Accurate entry/exit detection
- Functional prototype for real-time occupancy tracking

## Future Work
- IoT integration for cloud-based monitoring
- Mobile application support
- Live route-based occupancy display
